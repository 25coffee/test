静态版：编辑并导出（可选：用 GitHub Token 直接写回仓库的 docs/tests/tests.json）

1) 编辑测试

如果留空，会用 test_name 的 MD5 前 6 位自动生成（与原项目一致）。

2)（可选）直接写回 GitHub 仓库

GitHub Pages 本身不能写文件；但可以用 GitHub Contents API 提交一次 commit。 这需要一个有写权限的 Token（建议最小权限：Fine-grained token，授权到该 repo 的 Contents: Read and write）。
推送成功后，等 1–5 分钟 GitHub Pages 会自动更新。
